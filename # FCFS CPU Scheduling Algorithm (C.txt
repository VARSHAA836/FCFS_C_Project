# FCFS CPU Scheduling Algorithm (C Program)

Description
This project implements the **First Come, First Served (FCFS)** CPU scheduling algorithm using the C programming language.

FCFS is a non-preemptive scheduling algorithm where processes are executed in the order they arrive in the ready queue (FIFO ‚Äì First In, First Out).

- No priority or time slicing is involved.
